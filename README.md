# am-sensor-website

Website for AM - Sensor app

## Structure

```
am-sensor-website/
├── src/

│   ├── index.html        # Main HTML file with React

├── public/
│   └── styles.css        # Compiled Tailwind CSS output
├── tailwind.config.js    # Tailwind configuration
├── package.json          # Node.js project configuration
└── README.md
```
